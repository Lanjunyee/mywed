# 我的个人网站

一个用纯 HTML/CSS/JS 构建的极简个人网站。  
设计风格受 [cameronmcefee.com](https://cameronmcefee.com/) 启发，注重内容优先和干净整洁的视觉体验。

## 功能

- **首页** — 头像、姓名、一句话简介、社交链接（GitHub、邮箱）
- **项目** — 项目展示卡片，包含标题、描述和技术栈标签
- **博客** — Coming Soon 占位页面

## 技术栈

- HTML5
- CSS3（Flexbox 弹性布局、响应式设计、系统字体）
- 原生 JavaScript
- 零框架，零依赖

## 项目结构

```
.
├── index.html        # 首页
├── projects.html     # 项目页
├── blog.html         # 博客页
├── css/
│   └── style.css     # 全局样式
├── js/
│   └── main.js       # 导航交互
└── README.md
```
